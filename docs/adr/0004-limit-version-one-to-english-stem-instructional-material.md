# Limit version one to English STEM instructional material

Version 1 will support English-language STEM lecture notes and instructional material containing printed or handwritten prose, equations, diagrams, and tables, using the supplied current-and-resistance notes as an exemplar. Novels, forms, newspapers, music, multilingual documents, and other document classes require different semantic models; they may be processed experimentally but must produce an unsupported-input Conversion Warning rather than an unsupported accessibility claim.
