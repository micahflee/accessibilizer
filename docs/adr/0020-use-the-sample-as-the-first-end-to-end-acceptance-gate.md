# Use the sample as the first end-to-end acceptance gate

The first release milestone will use a manually authored gold Review Record for all 11 pages of the supplied electrical-current notes. The record may be drafted with assistance but becomes the acceptance oracle only after user approval, with genuinely ambiguous handwriting marked rather than guessed. Acceptance requires zero unflagged semantic errors against that record, warnings for actual ambiguities, visual equivalence within a small rendering tolerance, and clean veraPDF PDF/UA-1 validation. Routine development acceptance will exercise the gold output with macOS Preview and VoiceOver, and each release will also run a recorded manual checklist with Windows Adobe Acrobat Reader and NVDA; these are release tests, not per-conversion Reviewer steps. Performance on other STEM documents remains experimental until a larger representative corpus supports broader accuracy claims.
