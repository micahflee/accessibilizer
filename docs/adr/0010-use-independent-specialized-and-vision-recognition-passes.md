# Use independent specialized and vision recognition passes

Version 1 will run specialized local recognition for layout, handwriting, formulas, and tables, and extract any pre-existing PDF text and geometry as an additional non-authoritative candidate. It will then give each rendered page and those candidates to a vision-capable LLM for semantic reconstruction, Logical Reading Order, Spoken Math Alternatives, and figure descriptions. Formulas, tables, disputed regions, and every complex figure receive additional high-resolution calls; complex figures use a distinct crop-level prompt whose interpretation is reconciled with the page description, recognized labels, and geometry. Agreement supports automatic acceptance, while disagreement or ambiguity produces a Conversion Warning; no existing text layer or LLM proposal may silently overwrite disputed source content. Schema-valid model responses are cached in the Conversion Bundle to avoid repeated cost.
